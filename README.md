##  react-custom-template 
 - 基于creacte-react-app 的自定义react项目脚手架
## feature
- react 16.4.1
 - react-router 4.3.1
 - react-redux 5.0.7
 - 父子组件+路由传参
 - 状态管理
 - ...

## Build Setup
    # install dependencies
    npm install / yarn
    
    # serve with hot reload at localhost:8080
    npm start / yarn start
    
    # build for production with minification
    npm run build
